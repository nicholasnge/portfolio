Settings

Portfolio data is stored as data.json in your GitHub repo and served via GitHub Pages. Anyone with the link can view; only devices with your Personal Access Token can edit.

Display
GitHub

When creating the token: select your repo under Repository access, then grant Contents → Read & Write. The token is only stored in your browser's localStorage and never sent anywhere except GitHub.

Deploy to GitHub Pages
1. Create a public GitHub repo (e.g. "portfolio") 2. Upload index.html to the repo root 3. Settings → Pages → Source: Deploy from branch main / root → Save 4. Your public URL: https://<username>.github.io/<repo>/ 5. Enter your GitHub username, repo, and PAT above 6. Add positions — first save creates data.json automatically
